Personal Projects

Electric Couch Vehicle

Electric driving couch built from scratch. Custom 72V 40Ah Li-ion battery from 18650 cells; 5 Miles per charge; 15 MPH top speed; Controlled by Bluetooth PS4 Controller; Programmed in C++

Developing Collaborative Mixed Reality Environments

This project focuses on advancing the state of collaborative mixed reality (MR) environments. It integrates Microsoft's Mixed Reality Toolkit (MRTK2) to enable stable and immersive collaborative MR experiences, while recording over 50 positional and rotational metrics every 100 milliseconds.

HackUMass X Winner: Hacking RFID

36 Hour Hackathon project: Reverse engineered and reimplemented Wiegand protocol, used in ~40% of RFID systems, through bit-banging. Designed an injectable, "man in the middle", sniffing device. Created a GUI for demonstrating data coming across data lines. Demonstrated remote replay and DoS attacks live

Email Text-to-MP3 Summarization Tool

A Python application that will summarize your unread emails over a specified time interval. The summarization will be sent as an text message containing an MP3 file back to the user, allowing them to listen to their unread emails. Utilizes Google API, Twilio API & ChatGPT API

Temperature Access Control System | Atmega328P

Access control system that detects people, checks their temperature, and allows/denies entry on a door lock. The system is built upon an Atmega328p, programmed in Ansi C, with an OLED display, 2 Ultrasonic sensors, a DC motor, TMP36 temperature sensor, and 7-segment-LED.

Node.JS Reactive Personal Website

NVIDIA Jetson Machine Learning
